# App Store Assets Guide

This document outlines the assets needed for app store listings.

## Required Assets

### App Icons

#### Android (Google Play)
- **512x512 px** - High-res icon (PNG, no transparency)
- **1024x1024 px** - Feature graphic (optional)

#### iOS (App Store)
- **1024x1024 px** - App icon (PNG, no transparency, no alpha channel)
- Multiple sizes for different devices (handled by Xcode)

### Screenshots

#### Android
- **Phone**: At least 2 screenshots (up to 8)
  - Minimum: 320px height
  - Maximum: 3840px height
  - Aspect ratio: 16:9 or 9:16
- **Tablet** (optional): 7" and 10" screenshots

#### iOS
- **iPhone**: 
  - 6.7" display (iPhone 14 Pro Max): 1290 x 2796 px
  - 6.5" display (iPhone 11 Pro Max): 1242 x 2688 px
  - 5.5" display (iPhone 8 Plus): 1242 x 2208 px
- **iPad** (optional):
  - 12.9" iPad Pro: 2048 x 2732 px
  - 11" iPad Pro: 1668 x 2388 px

## App Store Listing Checklist

### Android (Google Play)
- [ ] App icon (512x512)
- [ ] Feature graphic (1024x500) - optional
- [ ] Phone screenshots (at least 2)
- [ ] Tablet screenshots (optional)
- [ ] Short description (80 chars)
- [ ] Full description
- [ ] Category selection
- [ ] Content rating questionnaire
- [ ] Privacy policy URL
- [ ] Contact email
- [ ] App signing setup

### iOS (App Store)
- [ ] App icon (1024x1024)
- [ ] iPhone screenshots (all required sizes)
- [ ] iPad screenshots (if supporting iPad)
- [ ] App name (30 characters)
- [ ] Subtitle (30 characters)
- [ ] Description (4000 characters)
- [ ] Keywords (100 characters)
- [ ] Category selection
- [ ] Privacy policy URL
- [ ] Support URL
- [ ] Age rating
- [ ] App review information

## Creating Screenshots

1. Run the app on a device/emulator
2. Navigate to key screens:
   - Login screen
   - Dashboard
   - Residents list
   - Create resident form
   - Complaints screen
   - Finance screen
3. Take screenshots using device tools
4. Edit screenshots to add:
   - Device frame (optional)
   - App branding
   - Feature highlights
5. Save in required formats and sizes

## Tools for Asset Creation

- **Screenshot Tools**: 
  - Android Studio Device Manager
  - Xcode Simulator
  - Fastlane (automated)
- **Image Editing**:
  - Figma
  - Adobe Photoshop
  - Canva
- **App Icon Generators**:
  - AppIcon.co
  - IconKitchen

## Notes

- All images must be high quality
- No placeholder text in screenshots
- Show real app functionality
- Highlight key features
- Keep descriptions clear and concise
- Update assets when app features change
